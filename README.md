# 三国杀面杀体验优化工具 🎲

专为三国杀面杀场景设计的武将抽取辅助工具，解决武将数量多、抽取繁琐、强度失衡及携带不便的痛点，让面杀更流畅、公平、有趣。

## ✨ 核心功能亮点

- **双模式适配**：支持普通模式（2-8人，每人抽取5名武将）与斗地主模式（抽取5+3+3名武将），覆盖主流面杀玩法
- **全扩展池覆盖**：集成以下官方扩展包武将（数据源自三国杀移动版wiki）：
  - 标准包
  - 神话再临·风/林/火/山
  - 界限突破·标准/林/火/山
  - 一将成名2011-2015
  - 界一将成名2012-2015
- **智能禁用机制**：支持自定义禁用武将，灵活适配面杀规则与强度平衡需求
- **结果一键保存**：自动记录当次抽取结果，便于复盘、分享或后续复用
- **数据权威可靠**：武将信息、技能描述及强度参考均来自官方社区维护的wiki，确保准确性

## 🚀 使用说明

1.  **选择模式**：进入系统后，选择「普通模式」或「斗地主模式」，并设置参与人数
2.  **配置池选项**：勾选需要启用的武将扩展包（支持多选，未勾选则不纳入抽取池）
3.  **禁用武将（可选）**：在禁用列表中添加需要排除的武将，避免强度失衡
4.  **执行抽取**：点击「抽取」按钮，系统将随机生成符合规则的武将列表
5.  **保存结果**：一键保存抽取结果，用于面杀分配或后续复盘
